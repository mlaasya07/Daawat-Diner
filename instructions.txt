دعوت DINER - Restaurant Discovery Application
=====================================================

PROJECT OVERVIEW
================
دعوت Diner is a sophisticated restaurant discovery application that helps users find the perfect dining experience across major Indian cities. The app features a retro aesthetic with Arabic-inspired branding and provides intelligent restaurant recommendations based on user preferences.

FEATURES
========
✅ City-Area Based Search (12+ cities, 15+ areas each)
✅ Top 5 Restaurant Results per search
✅ Star System with non-repetitive refresh
✅ Custom hovering number keypad
✅ Real-time location-based filtering
✅ Cuisine-specific restaurant matching
✅ Age-based dietary restrictions
✅ Budget calculations and filtering
✅ Satirical error messages for exhausted searches
✅ Responsive design with retro color palette
✅ Direct integration with Google Maps

CORE FUNCTIONALITY
==================

1. RESTAURANT SEARCH
   - Input: Number of people, ages, cuisines, diet preference, budget, city, area
   - Output: Top 5 restaurants with ratings, distance, price level, contact info
   - Algorithm: Geoapify API + fallback system for comprehensive coverage

2. STAR SYSTEM
   - Users can star restaurants to lock them in results
   - Refresh replaces only non-starred restaurants
   - Warning when all 5 restaurants are starred
   - Non-repetitive results across searches

3. NUMBER KEYPAD
   - Custom hovering keypad for number inputs
   - Prevents page scrolling during use
   - Brand-colored design with proper button layout
   - Works for: number of people, ages, budget

4. LOCATION INTELLIGENCE
   - Two-step city-area selection
   - 3km radius search within selected area
   - Real coordinates from geocoding API
   - Accurate Google Maps integration

5. SATIRICAL ERROR HANDLING
   - Humorous messages when search options are exhausted:
     * "Try other cuisines, you indecisive human! 🙄"
     * "Seriously? We've shown you everything in this area. Pick something! 😤"
     * "Maybe try a different area? This one's exhausted! 🤷‍♂️"
     * "Are you planning to eat at ALL the restaurants? Choose one already! 🍽️"
     * "Even Google Maps is tired of your indecisiveness! 🗺️"
     * "Plot twist: Maybe cook at home? 🏠👨‍🍳"
     * "Breaking news: Local person can't decide where to eat! 📰"
     * "Time to expand your culinary horizons! 🌍"

TROUBLESHOOTING
===============

Common Issues:
1. No restaurants found: Try different cuisine combinations or areas
2. API rate limits: Built-in caching reduces API calls
3. Location not found: Verify city-area combination exists
4. Keypad not appearing: Check for JavaScript errors in console

Support:
- Check browser console for error messages
- Verify internet connection for API calls
- Ensure JavaScript is enabled
- Clear browser cache if experiencing issues
